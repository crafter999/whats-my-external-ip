# whats-my-external-ip

A CLI tool to get your public IP address with **0 dependencies** using the following services:
- https://checkip.amazonaws.com
- https://api.ipify.org
- https://ipinfo.io/ip

## Installation

```bash
npm install -g whats-my-external-ip
```

## Usage

```bash
whats-my-external-ip # returns your public IP address
```
